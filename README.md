# Python Voice Assistant (Jarvis Inspired)

A voice-controlled AI assistant built using Python.
It can perform basic tasks using voice commands.

## Features
- Voice command recognition
- Open YouTube
- Open Calculator
- Tell jokes
- Basic system automation

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Webbrowser
- OS module

## How to Run
1. Install required libraries
2. Run main.py
3. Speak commands

## Note
This project is for educational and learning purposes.
